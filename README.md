# gamma-NaAsSe2_Symmetry-Mode-Analysis

This repo contains scripts, structure files, and results from symmetry-mode analyis of gamma-NaAsSe2. Further details are provided below.

- Analysis: contains scripts used for motif geometry quantification, projected density-of-states (PDOS) plotting, and second-harmonic generation (SHG) data plotting.

- Results: contains results from geometry analysis of structure files (Geometry Metrics) and ab initio simulation outputs for pressure and full distortion experiments (Simulation Outputs).

- Simulations: contains scripts used for ab initio simulations, such as parameter convergence, structural relaxation, electronic ground state, and SHG calculations.

- Structures: contains .cif structure files for the As/Se projected distortion (As-Se Distortion), the full distortion (Full Distortion), relaxed structures as a function of pressure (Pressure), and primary parent/child structures used as the starting point for symmetry-mode analysis (Primary).
